# 🎓 Next.js 초보자를 위한 학습 가이드

안녕하세요! 이 가이드는 **Next.js App Router를 처음 배우는 분들**을 위해 만들어졌습니다.

## 🤔 Next.js가 뭔가요?

**레스토랑 비유로 이해하기:**

상상해보세요. 당신이 레스토랑을 운영한다고 생각해봅시다.

- **주방 (서버)**: 손님이 보지 못하는 곳에서 음식을 준비합니다
- **테이블 (클라이언트/브라우저)**: 손님이 앉아서 음식을 먹고 반응하는 곳입니다
- **종업원 (API)**: 주문을 받아서 주방에 전달하고, 완성된 음식을 가져옵니다

Next.js는 이 **레스토랑 전체를 관리**하는 시스템입니다!

### Server Component (주방 👨‍🍳)
- 손님이 보지 못하는 곳에서 일합니다
- 비밀 레시피(API 키, DB 비밀번호)를 안전하게 보관합니다
- 음식(데이터)을 미리 준비해서 빠르게 제공합니다
- **예시**: 페이지를 처음 열 때 데이터베이스에서 제품 목록 가져오기

### Client Component (테이블 🍽️)
- 손님이 직접 보고 상호작용하는 곳입니다
- 버튼을 누르거나, 입력을 받거나, 화면을 업데이트합니다
- 종업원(API)을 불러서 추가 주문을 할 수 있습니다
- **예시**: 삭제 버튼 클릭, 검색어 입력, 정렬 변경

### API Routes (주문서 📋)
- 손님과 주방 사이의 소통 수단입니다
- 주문을 받아서 주방에 전달하고, 완성된 음식을 가져옵니다
- 잘못된 주문은 거절합니다
- **예시**: "제품 목록 주세요", "이 제품을 삭제해주세요"

## 📚 이 가이드의 특징

### ✨ 초등학생도 이해할 수 있어요
- 전문 용어를 최대한 피했습니다
- 어려운 개념은 **쉬운 비유**로 설명합니다
- 예시: "Server Component" → "주방", "Client Component" → "테이블"

### 🎯 단계별로 따라가기 쉬워요
- 코드가 실행되는 순서를 **Step 1, 2, 3...**으로 설명합니다
- 각 단계마다 **파일 위치**와 **줄 번호**를 알려줍니다
- 실제 코드를 직접 열어서 확인할 수 있습니다

### 💡 실전에 바로 쓸 수 있어요
- **복사해서 붙여넣기** 할 수 있는 코드 템플릿 제공
- 다른 프로젝트에도 똑같이 적용 가능
- 체크리스트로 빠뜨리는 것 없이 작업 가능

### ✏️ 직접 해보며 배워요
- 읽기만 하지 않고 **직접 코드를 작성**합니다
- 실습 과제가 단계별로 제공됩니다
- 완료하면 스스로 기능을 만들 수 있게 됩니다

## 📂 이 프로젝트 구조 (간단 버전)

```
nextjs-trm-crud-testdb/
│
├── src/app/                    ← 페이지들 (라우팅)
│   ├── page.tsx               ← 메인 페이지
│   ├── trm/target-products/   ← 제품 관리 페이지
│   └── api/                   ← API 엔드포인트
│
├── src/components/            ← 재사용 컴포넌트
├── src/lib/
│   ├── services/              ← 비즈니스 로직
│   ├── types/                 ← 타입 정의
│   ├── utils/                 ← 유틸리티 함수
│   └── hooks/                 ← 커스텀 훅
│
└── prisma/                    ← 데이터베이스 설정
    └── schema.prisma          ← DB 스키마
```
