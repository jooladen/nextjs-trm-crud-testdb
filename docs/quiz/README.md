**완료 기준**:

## 🎯 문제 유형

### 객관식 (4지선다)
명확한 개념 이해를 확인합니다.
```
문제: Server Component의 특징은?
A) 'use client'가 필요하다
B) async 함수로 만들 수 있다
C) useState를 사용할 수 있다
D) 브라우저에서만 실행된다
```

### 빈칸 채우기
코드 문맥에서 정확한 용어를 알고 있는지 확인합니다.
```
Server Component에서 데이터를 가져올 때는
_____ 함수를 사용합니다.
```

### 코드 분석
실제 코드를 읽고 이해하는 능력을 테스트합니다.
```
다음 코드의 실행 순서는?
[코드 제시]
```

### 실습/서술형
실전 시나리오에서 문제를 해결하는 능력을 평가합니다.
```
N+1 쿼리 문제를 해결하는 코드를 작성하세요.
```

## ✅ 완료 기준

### 전체 퀴즈 시스템 완료 시

**기본 능력**:
- ✅ Next.js App Router 아키텍처를 이해한다
- ✅ Server/Client Component를 적절히 사용한다
- ✅ CRUD를 처음부터 구현할 수 있다

**실전 능력**:
- ✅ 에러를 빠르게 디버깅할 수 있다
- ✅ 성능 문제를 식별하고 최적화한다
- ✅ 패턴을 다른 프로젝트에 적용할 수 있다

**전문가 수준**:
- ✅ 새로운 기능을 설계부터 구현까지 완수한다
- ✅ 팀원에게 아키텍처를 명확히 설명한다
- ✅ 코드 리뷰에서 개선점을 제안한다
